使用 MedTimer 掌握您的用藥管理

MedTimer 是一款開源的用藥提醒應用程式，協助您有效管理藥物與藥丸，同時完全掌控個人敏感資料。

靈活且個人化的提醒功能：

- 管理不限數量的藥物，並可為每種藥物設定自訂提醒（也可使用每日提醒的簡易預設）。
- 建立具彈性的提醒，可設定休息期間與特定服藥日（如避孕藥）。
- 週末模式：可在選定的日子延後提醒時間。
- 可延後（Snooze）通知或設定重複提醒。
- 即時新增額外劑量。
- 建立間隔式提醒或依序間隔的連續提醒。

簡單又安全的資料管理：

- 可確認或忽略提醒，以準確記錄服藥狀況。
- 提供圖表與表格檢視過去的服藥紀錄。
- 可將用藥歷史匯出為 CSV 檔，方便保存或與醫療專業人員分享。
- 支援以 JSON 檔備份與還原用藥清單。
- 提供藥品庫存追蹤與庫存不足提醒。
- 可為藥品加上標籤並進行篩選。

隱私與離線存取：

- 所有資料皆安全儲存在您的裝置中，確保完全的隱私與離線可用性。
- 無需網路連線，您的用藥提醒隨時可用。

MedTimer 為免費、無廣告的應用程式。
